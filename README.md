## Jalankan Server secara Lokal

```
npm i -g pnpm

pnpm i
pnpm run dev
```

Lalu kunjungi `http://localhost:3000`

Atau pasang [Ekstensi Vite untuk VS Code](https://marketplace.visualstudio.com/items?itemName=antfu.vite) untuk menyunting secara berdampingan.
